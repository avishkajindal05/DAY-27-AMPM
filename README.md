## 🎯 Problem Statement

The dataset contains multiple real-world issues affecting business operations, including:

* Outliers in order values
* Missing or incorrect delivery times
* Data entry errors in preparation time
* Structural inconsistencies in customer ratings

The goal is to:

* Identify these issues
* Understand their business impact
* Propose effective solutions

---

## 🔍 Key Insights & Solutions

### 1. Order Value Outliers

* **Issue:** Extreme values distort revenue metrics
* **Solution:** Apply capping using IQR or percentile methods

---

### 2. Delivery Time Issues

* **Issue:** Null or zero values
* **Solution:**

  * Replace with median/mean
  * Or remove invalid records

---

### 3. Preparation Time Errors

* **Issue:** Incorrect or unrealistic values due to manual entry
* **Solution:**

  * Convert to absolute values
  * Apply logical constraints

---

### 4. Customer Rating (Structural Issue)

* **Issue:** Inconsistent or biased rating system
* **Solution:**

  * Normalize ratings
  * Handle missing or invalid entries
